# NFC 芯片与 Vercel URL 集成指南

## Vercel 是否可以生成 URL 链接植入 NFC 芯片？

**答案：可以！但需要分开理解两个概念**

### 概念说明

1. **Vercel 的作用**
   - Vercel 是一个部署平台，负责托管您的网站
   - 部署成功后，Vercel 会提供一个 URL（域名）
   - 例如：`https://your-project.vercel.app`

2. **NFC 芯片的作用**
   - NFC（Near Field Communication）是一种短距离无线通信技术
   - NFC 芯片可以存储数据，包括 URL 链接
   - 当手机靠近 NFC 芯片时，会自动读取并跳转到存储的 URL

### 完整流程

```
1. 在 Vercel 部署网站 → 获得 URL（如 https://time-station.vercel.app）
                                ↓
2. 将这个 URL 写入 NFC 芯片 → 芯片存储 URL
                                ↓
3. 制作 NFC 卡片/标签 → 用户可以使用
                                ↓
4. 手机靠近 NFC → 自动跳转到您的网站
```

## 详细实施步骤

### 第 1 步：在 Vercel 部署项目获得 URL

参考 `VERCEL_DEPLOYMENT.md` 完成部署后，您会获得：

```
✅ Vercel 自动生成的域名：https://your-project-name.vercel.app
✅ 或者您的自定义域名：https://www.your-domain.com
```

**示例**：
```
https://time-station.vercel.app
```

### 第 2 步：准备 NFC 芯片/卡片

#### NFC 卡片类型

| 类型 | 容量 | 价格 | 用途 |
|------|------|------|------|
| **NTAG213** | 144 字节 | ¥0.5-1 | 存储 URL（推荐） |
| **NTAG215** | 504 字节 | ¥1-2 | 存储 URL + 小数据 |
| **NTAG216** | 888 字节 | ¥2-3 | 存储更多数据 |
| **Mifare Classic** | 1KB | ¥1-3 | 需要 NFC 写卡器 |

**推荐**: NTAG213 足够存储一个 URL，成本最低。

#### 购买渠道

- 淘宝/京东/拼多多：搜索 "NFC 卡片" 或 "NFC 标签"
- 电商平台：选择商家带写卡服务的会更方便

### 第 3 步：将 URL 写入 NFC 芯片

#### 方法一：使用手机（最简单）

**需要**：
- 一台支持 NFC 的手机（大部分智能手机都支持）
- NFC 写卡 App

**推荐 App**：

| App 名称 | 平台 | 特点 |
|----------|------|------|
| **NFC Tools** | iOS/Android | 免费、简单、功能齐全 |
| **TagWriter** | Android | 免费、NXP 官方 |
| **NFC Writer** | iOS | 免费、界面简洁 |

**操作步骤（以 NFC Tools 为例）**：

1. 下载并打开 NFC Tools App
2. 选择 **"Write"** 标签
3. 选择 **"Add a record"** → **"URL"**
4. 输入您的 Vercel URL：
   ```
   https://time-station.vercel.app
   ```
5. 选择 **"Write"** 标签
6. 将 NFC 卡片贴近手机背面
7. 等待提示 "Write successful"
8. 完成！

#### 方法二：使用 NFC 写卡器（专业方式）

**设备**：
- NFC 写卡器（ACR122U 等）
- 电脑

**软件**：
- MifareOne Tool
- NFC Tools PC

**步骤**：
1. 将 NFC 写卡器连接到电脑
2. 打开写卡软件
3. 读取卡片信息
4. 选择写入 URL 记录
5. 写入成功

### 第 4 步：测试 NFC 卡片

1. **关闭 NFC Tools** 或其他写卡 App
2. **打开手机 NFC 功能**（iOS 默认开启，Android 需在设置中开启）
3. **将手机贴近 NFC 卡片**
4. **手机会弹出提示**：是否打开链接？
5. **点击打开**，自动跳转到您的 Vercel 网站

✅ 如果能成功跳转，说明 NFC 写入成功！

## 技术限制和注意事项

### 1. NFC 芯片限制

| 限制 | 说明 |
|------|------|
| **容量** | NTAG213 只有 144 字节，URL 不能太长 |
| **写入次数** | 通常 10 万次以上，基本够用 |
| **读出次数** | 无限次 |
| **加密** | NTAG213 不支持加密，其他型号支持 |

### 2. 手机限制

| 限制 | 说明 |
|------|------|
| **NFC 功能** | 需要手机支持 NFC（大部分智能手机都支持） |
| **系统版本** | iOS 11+、Android 4.4+ |
| **权限** | Android 可能需要开启 NFC 权限 |
| **距离** | 贴合距离 2-5cm |

### 3. URL 限制

| 建议 | 说明 |
|------|------|
| **使用 HTTPS** | Vercel 自动提供 HTTPS |
| **URL 长度** | 控制在 50 字符以内 |
| **可读性** | 使用短链接更友好 |

## 常见问题

### Q1：NFC 卡片可以存储多个 URL 吗？

**A**：可以，但不推荐。存储多个 URL 会降低用户体验，建议每个卡片对应一个服务页面。

### Q2：iPhone 可以使用 NFC 吗？

**A**：可以！iOS 11+ 系统支持 NFC，但需要 App 触发（iOS 13+ 支持后台读取）。

### Q3：NFC 卡片会被恶意写入吗？

**A**：如果使用 NTAG216 或加密型号，可以设置密码保护。NTAG213 不支持加密，理论上可被覆盖写入。

### Q4：Vercel URL 可以更换吗？

**A**：项目名可以更改，但 Vercel 生成的 vercel.app 域名固定。推荐使用自定义域名或短链接。

### Q5：NFC 卡片防水吗？

**A**：取决于卡片材质。PVC 材质有一定防水性，但不建议长时间浸泡。
